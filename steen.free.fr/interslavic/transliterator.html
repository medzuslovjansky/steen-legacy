Notes:

  • This transliteration program works on a transliterate-as-you-type base. It transliterates Interslavic text from basic Latin to basic Cyrillic orthography and vice versa. If you need more options, try the extended transliterator, which looks almost the same, but can also transcribe Interslavic text into several alternative writing systems (taking the etymological alphabet as a starting point), flavourised versions, national orthographies and IPA.
  • If a Latin or Cyrillic letter is missing on your keyboard, you can insert them by clicking the small buttons between the input and the output field.
  • Alternatively, you can type either cz/cx for č, sz/sx for š, zs/zx/ż for ž, and ye for ě. If you are typing in Cyrillic, й, я, ль etc. are automatically converted to ј, ја, љ...
  • To prevent a fragment of text from being transliterated, place it between { and }. The accolades will automatically be removed afterwards. This feature can be convenient if you want to preserve the original spelling of e.g. a name in the transliterated text.
  • Text fragments between < and > are left unchanged, too. Useful for transliterating html code without changing the tags.
