Notes:

  • This transliteration program works on a transliterate-as-you-type base. It is an extension of the transliterator for standard Interslavic. It can do exactly the same as the latter, but it can also transcribe text from the scientific alphabet into various non-standard orthographies, including language-specific national orthographies, non-Slavic orthographies, and IPA.
  • To write non-ASCII characters, you can select them from the list between the input and the output box.
  • If you want to prevent a fragment of text from being altered, place it between { and }. The accolades will automatically be removed afterwards. This feature can be convenient if you want to preserve the original spelling of e.g. a name in the transliterated text.
  • Text fragments between < and > are left unchanged, too. Useful for transliterating html code without changing the tags.
